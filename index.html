About the Project

The inspiration for this project came from my time as a student where I noticed how difficult it was for institutions to efficiently manage student data. I wanted to create a solution that would simplify the process for academic staff while also providing valuable insights to educators and administrators. This project started as a personal challenge and eventually grew into something I could showcase as a Portfolio Project for Holberton School. The project was developed over a timeline of 4 months with continuous improvement based on feedback from users and mentors.
